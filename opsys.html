Chapter 3 Terminology Table

Terms Definitions
Operating System The software that controls the overall operation of a computer.
Job Queue Where jobs residing in mass storage, in batch processing systems, wait for execution.
FIFO First-in, first-out order.
Interactive Processing A feature that allows a program being executed to carry on a dialogue with the user through remote terminals.
Time-sharing Operating systems that provide service to multiple users at the same time.
Multiprogramming Time is divided into intervals and then the execution of each job is restricted to only one interval at a time.
Application Software Consists of the programs for performing tasks particular to the machine’s utilization.
System Software Performs those tasks that are common to computer systems in general.
Utility Software One of two catagories within the class of system software, which consists of software units.
Graphical User Interface (GUI) In which objects to be manipulated, such as files and programs, are represented pictorially on the display as icons.
Shell An older user interface which are the portion of an operating system that handles communication with the user.
Kernel The internal part of an operating system.
Device Driver The software units that communicate with the controllers (or at times, directly with peripheral devices) to carry out operations on the peripheral devices attached to the machine. A component of the kernel.
File Manager A unit whose job is to coorinate the use of the machine's mass storage facilities.
Memory Manager Is charged with the task of coordinating the machine’s use of main memory. Another component of an operating system's kernel.
Virtual Memory A large “fictional” memory space created by paging.
Boot Strapping Transfers the operating system from mass storage (where it is permanently stored) into main memory. Preformed by a computer everytime it is turned on.
ROM Read-only memory, its contents can be read but not altered.
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling The technique of holding data for output at a later but more convenient time.
Super User (or administrator) A person who establishes accounts. They gain highly privileged access to the operating system.
Multi-factor Authentication Requires more than one piece of evidence to gain access to a computer account.
