Chapter 4 Terms:

Terms Definitions
Network Classifications:
LAN Local area network: classification of a computer network.
MAN Metropolitan area network: classification of a computer network.
WAN Wide area network: classification of a computer network.
Network Models:
bus network The machines are all connected to a common communication line called a bus.
star network A large central computer serving many users. the star configuration is also popular in wireless networks where communication is conducted by means of radio broadcast.
Protocols:
CSMA/CD Ethernet Carrier Sense, Multiple Access with Collision Detection: a protocol which controls the right to transmit messages in a bus network.
CSMA/CA Wifi Carrier Sense, Multiple Access with Collision Avoidance: a policy wireless networks adopt of trying to avoid collisions rather than detect them.
Combining Networks:
repeater A device that passes signals back and forth between the two original buses (usually with some form of amplification) without considering the meaning of the signals.
bridge Connects two buses, it looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side.
switch Essentially a bridge with multiple connections.
router Handles the connection between networks to form an internet.
gateway The “point” at which one network is linked to an internet.
Client/server model:
P2P Peer-to-peer - involves processes that provide service to and receive service from each other.
distributed system They consist of software units that execute as processes on different computers.
Cloud computing Huge pools of shared computers on the network can be allocated for use by clients as needed.
Internet Addressing:
Tier 1 Very high-speed, high-capacity, international WANs. Thought of as the backbone of the Internet. Essentially networks of routers that collectively provide the Internet’s communication infrastructure.
Tier 2 Connect to the tier-1 ISPs. Tend to be more regional. Essentially networks of routers that collectively provide the Internet’s communication infrastructure.
Tier 3 An intermediary, also called an access. Provides access to the network core.
IP address:
ICANN (https://www.icann.org/)
DNS
TLD Top-level domain
Country-code TLDs (https://www.worldstandards.eu/other/tlds/)
Internet Protocols:
FTP File Transfer Protocol, an application for listing and copying files across the network.
HTTP Hyper Text Transfer Protocol, Hypertext documents are normally transferred between browsers and webservers using this protocol.
SMTP Simple Mail Transfer Protocol, initially designed for transferring text messages encoded with ASCII.
World Wide Web:
URL Uniform Resource Locator, a unique address given to a document in order to locate and retrieve the documents on the Web.
client-side activities They are performed by a client (such as a browser).
server-side activities They are performed by a server (such as a webserver).
Internet four-layer software:
Application layer A message typically originates in the application layer, from theree it is passed down through the transport and network layers.
Transport layer A messsage is passed down from the application layer as it is prepared for transmission.
Network layer A Message is passed down through this layer as it is prepared for transmission.
Link layer The message is recieved by this layer and finally it is transmitted.
Internet is:
Packet-switching network Digital data is sent by breaking the data into blocks of bits called packets.
Fault tolerant Routing on the Internet is fault tolerant and redundant.
Redundant The redundancy of routing (i.e., more than one way to route data) between two points on the Internet increases the reliability of the Internet and helps it scale to more devices and more people.
Cybersecurity
Malware Malicious software
Virus Software that infects a computer by inserting itself into programs that already reside in the machine.
Worm An autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
Trojan horse A Trojan horse is a program that enters a computer system disguised as a desirable program.
Spyware Software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack (sometimes called sniffing software).
Phishing A technique of obtaining information explicitly by simply asking for it.
DDoS When the work of generating the attacks is distributed over many hosts, this is known as a distributed denial of service (DDoS) attack.
