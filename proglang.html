Chapter 6 Programming Languages

Source Program
Terms Definitions
Machine Language An encoding system that consists of sequences of instructions that are encoded as numeric digits.
Debugging Locating errors in the code and correcting them.
Assembler Converts a program in mnemonic form into machine language.
Assembly Language A mnemonic system for representing programs collectively.
Translator Translats programs expressed in high-level primitives into machine-language programs.
Compiler Translation program.
Interpreter An alternative to translators, another means of implementing third-generation languages.
Cross-platform Software Software that is independent of an operating system’s design as well as the machine’s hardware design and is therefore executable throughout a network.
Natural Language Such as English, German, and Latin, evolved over time without formal grammatical analysis.
Formal Language Such as programming languages, precisely defined by grammars.
Machine Independence Do not rely on the characteristics of a particular machine.
High-level Programming Language They make computer programs both easier for human authors to write, and easier for subsequent readers to understand.
